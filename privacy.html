Privacy Policy

Last Updated: January 2026

Zinrex respects user privacy. This Privacy Policy explains how personal data is collected, used, and protected in compliance with KVKK (Turkey) and GDPR (EU).

1. Data We Collect

  • Account Information: Username
  • Profile Data: Information voluntarily provided by users
  • User Content: Posts, comments, likes, and interactions
  • Technical Data: Device type, app version, crash logs (anonymous)

2. Purpose of Data Use

  • Account creation and authentication
  • Providing app features and social interactions
  • Security, fraud, and abuse prevention
  • Improving app performance and user experience
  • Legal and regulatory compliance

3. Data Storage and Retention

  • Data is securely stored using Google Firebase services
  • Personal data is retained only as long as necessary
  • Upon account deletion, all personal data is permanently removed within 30 days

4. Data Sharing

  • Zinrex does not sell personal data
  • Data is shared only when legally required
  • Limited data may be processed by service providers strictly for app functionality

5. User Rights

  • Access and update personal data
  • Request account and data deletion
  • Object to data processing where applicable

Requests can be submitted via: info@zinrex.com

6. Children’s Privacy

Zinrex is not intended for children under the age of 13. We do not knowingly collect personal data from children.

7. Security

  • All data transfers use SSL/HTTPS encryption
  • Technical and administrative safeguards are in place

8. Contact
